## Description

Frontend application build with nextjs 13 appDir and Typescript

## Installation

```bash
$ nvm use

$ npm install
```

## Running the app

```bash
# development
$ npm run dev
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

## Test

```bash
# unit tests
$ npm run test
```
